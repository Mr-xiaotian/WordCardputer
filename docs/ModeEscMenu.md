# ModeEscMenu.ino

## 概述
`ModeEscMenu.ino` 实现了应用程序的**全局暂停/控制菜单（ESC 菜单）**。用户在学习、听读或听写等主要模式中按下 `ESC`（键盘上的 `` ` `` 键）即可随时呼出该菜单。

## 核心内容
- **菜单项定义**：
  提供了一个静态列表 `escItems`，包含应用级别的所有核心操作指令：
  - 手动保存进度
  - 查看学习统计
  - 重新选择词库或语言
  - 在不同功能模式间切换（学习、听读、听写）
  - 查看按键帮助和 Web 控制台信息
- **菜单初始化与渲染 (`initEscMenuMode`, `drawEscMenu`)**：
  - 重置光标和滚动位置。
  - 调用通用绘制函数 `drawTextMenu` 在屏幕上渲染选项列表。
- **事件处理分发 (`loopEscMenuMode`)**：
  - **返回上一模式**：再次按下 `ESC` 键时，自动保存未落盘的数据（`autoSaveIfNeeded`），并无缝恢复到呼出菜单前的模式界面。
  - **导航**：处理 `;/.` 键的光标移动和视图滚动。
  - **执行命令**：处理 `Enter` 键按下的逻辑分发，根据选中的索引调用对应的功能函数（如进入 `MODE_STATS`，或切换回 `MODE_FILE_SELECT` 等）。
  - **Web 控制台提示**：特殊处理显示当前设备的局域网 IP 地址，提示用户如何从浏览器访问。

## 关联模块
- **中枢枢纽**：作为应用的控制中枢，它与几乎所有的模式模块都有交互跳转关系。
- **状态维护**：依赖全局变量 `previousMode` 来记忆从哪个界面进入菜单，以保证准确退出。
