# ModeStudy.ino

## 概述
`ModeStudy.ino` 实现了应用的核心功能——**双面闪卡学习模式**。它借鉴了 Anki 的间隔重复和正反面回忆机制，提供了一个高度互动的单词记忆界面，支持日语和英语两种语言体系。

## 核心内容
- **Anki 风格的双面设计 (`showAnkiSideA`, `showMeaning`)**：
  - **动态出题面**：每次抽取单词时，系统会随机（或根据特定逻辑）决定初始显示的“正面（Side A）”。
    - 如果是外语优先（`Side A`）：正面显示英语/日语原文，按键翻转后显示中文释义。
    - 如果是中文优先（`Side B`）：正面显示中文释义，要求用户回忆外语原文，按键翻转后核对答案。
  - **翻卡交互**：通过 `showMeaning` 布尔值控制当前是处于“提问”还是“答案”状态。
- **多语言渲染分发 (`drawStudyWord`, `drawJapaneseWord`, `drawEnglishWord`)**：
  - **日语卡片**：处理假名（`jp`）、汉字写法（`kanji`）、音调（`tone`）、词性（`pos`）和中文释义（`zh`）的精细排版。
  - **英语卡片**：处理英文原文（`en`）、词性（`pos`）、中文释义（`zh`），并特别处理音标（`phonetic`）——调用 `UtilsString.ino` 中的音译函数将 IPA 转化为可显示的 ASCII 字符。
  - **自适应排版**：广泛使用 `drawAutoFitString` 确保无论单词长短都能完美适配屏幕。
- **主交互循环 (`loopStudyMode`)**：
  - `Enter` 或 `Space` 键：执行翻卡动作（揭晓答案），如果已经是答案状态，则触发“认识（+1 分）”并抽取下一个词。
  - `1-5` 数字键：用户在翻卡后，可以手动为当前单词的熟练度打分。1 代表完全不认识（增加出现概率），5 代表完全掌握。打分后自动切到下一词，并触发 `markScoreDirty`。
  - `Up/Down` 键：实时调节系统音量。
  - `/` 或 `,` 键：主动请求重新播放当前单词的音频发音。
- **状态流转与入口 (`startStudyMode`)**：
  - 负责从文件选择模式接收到 JSON 文件路径后，调用 `loadWordsFromJSON` 加载数据。
  - 如果加载成功，重置索引，初始化第一个单词并开始学习循环；如果失败，则显示错误信息并退回主菜单。

## 关联模块
- **数据与存储**：依赖 `UtilsData.ino` 进行文件读取、评分更新、自动保存以及加权随机抽词。
- **音频播放**：依赖 `UtilsAudio.ino` 播放单词语音。
- **文本渲染**：依赖 `UtilsString.ino` 进行自适应文字和特殊符号的绘制。
