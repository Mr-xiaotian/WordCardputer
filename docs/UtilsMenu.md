# UtilsMenu.ino

## 概述
`UtilsMenu.ino` 是项目的**通用 UI 渲染工具库**，专门用于在 M5Cardputer 的屏幕上绘制标准化的菜单列表和数据表格，以确保整个应用在各个模块下的视觉风格一致。

## 核心内容
- **通用可滚动菜单 (`drawTextMenu`)**：
  - 这是一个高度参数化的渲染函数，被广泛应用于语言选择、文件选择和 ESC 菜单中。
  - **布局结构**：
    - 左上角：绿色标题。
    - 右上角：可选的电池电量百分比显示（通过 M5Cardputer API 获取）。
    - 主体区域：根据 `scrollIndex` 和 `visibleLines` 裁剪出的当前页项目列表。
    - 右下角：可选的“当前项/总数”分页指示器。
  - **交互反馈**：当前 `selectedIndex` 所指向的列表项会使用黄色高亮，并添加 `>` 前缀以明确指示焦点位置。
  - **异常处理**：当传入的列表为空时，居中显示红色的 `emptyText` 提示信息。
- **简易表格渲染 (`drawSimpleTable`)**：
  - 用于在“学习统计”等模式中展示结构化数据。
  - 接收表头数组、列坐标（X 轴位置集合）和二维数据行数组。
  - 自动绘制带有下划线分隔符的表头，并按设定的坐标对齐填充数据内容，确保数据展示清晰易读。

## 关联模块
- **被调用方**：`ModeLangSelect.ino`、`ModeFileSelect.ino`、`ModeEscMenu.ino` 和 `ModeStats.ino` 等直接调用这些函数构建页面。
- **依赖**：依赖 `M5GFX` 库的画布 (`M5Canvas`) API 进行底层绘图操作。
