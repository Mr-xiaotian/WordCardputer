# WordCardputer.ino

## 概述
`WordCardputer.ino` 是本项目的**主程序入口文件**。它定义了整个应用程序的全局变量、核心数据结构以及状态枚举，并实现了 Arduino 的标准生命周期函数 `setup()` 和 `loop()`。

## 核心内容
- **全局状态与配置**：包含当前的运行模式（`AppMode`）、语言选择（`StudyLanguage`）、文件路径、音量、亮度状态等。
- **数据结构**：定义了 `Word` 结构体，用于统一承载日语和英语的单词信息（如文本、释义、音标、词性、声调、熟练度等）；以及 `DictError` 结构体，用于记录听写错误。
- **硬件初始化 (`setup`)**：
  - 初始化 M5Cardputer 硬件、串口、扬声器。
  - 初始化 SPI 总线并挂载 SD 卡。
  - 尝试读取环境变量连接 WiFi，并在成功后启动 Web 服务器和 NTP 时间同步。
  - 创建双缓冲画布并初始化屏幕。
- **主循环 (`loop`)**：
  - 刷新硬件状态。
  - 根据 `appMode` 将逻辑分发到各个子模式的 `loop` 函数。
  - 处理后台 Web 服务器请求。
  - 实现自动亮度控制与省电休眠机制。

## 关联模块
- **模式模块**：将具体的业务逻辑交由各个 `Mode*.ino` 文件处理。
- **工具模块**：通过 `Utils*.ino` 提供的辅助函数进行绘图、数据存储、网络通信等操作。
