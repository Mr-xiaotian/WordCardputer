# UtilsString.ino

## 概述
`UtilsString.ino` 是项目的**字符串处理与高级文本绘制工具库**。它补充了 `M5GFX` 库中缺失的自适应文本布局功能，并解决了特定字符集的渲染难题。

## 核心内容
- **自适应文本绘制 (`drawAutoFitString`)**：
  - 用于处理长度不一的单词或长句（如较长的英文单词或中文释义）。
  - **核心算法**：接受一个初始基准字号（如 2.0），通过 `textWidth` 方法预计算文本占据的像素宽度。如果宽度超出了屏幕的安全显示区域，则通过循环每次将字号缩小 0.1，直到文本能够完整放下或触及设定的最小字号（0.8）。这确保了长文本不会被截断而影响阅读。
- **角标绘制助手 (`drawTopLeftText`, `drawTopRightText`)**：
  - 简化在屏幕特定位置（如左上角的学习进度、右上角的电池电量）绘制状态信息的逻辑，内部封装了对齐方式（Datum）和坐标偏移计算。
- **音标音译转换 (`asciiPhonetic`)**：
  - **背景**：内置的中文字体库（如 `efontCN`）通常不包含 IPA（国际音标）的 Unicode 扩展字符，导致英文音标渲染成空白或乱码。
  - **解决方案**：实现了一个基于规则的查表替换函数。它解析 UTF-8 编码字节流，捕捉常见的 IPA 音标多字节组合（如 `θ`, `ð`, `ʃ`, `ʒ`, `æ` 等），并将它们降级转换为最接近的 ASCII 字母组合（如 `th`, `sh`, `ae`）。这种轻量级的处理方式避免了引入庞大的专用音标字库，节省了宝贵的 Flash 空间。

## 关联模块
- 主要服务于 `ModeStudy.ino` 中的卡片绘制，特别是对英文单词和音标的渲染。
