# Agents

## 环境配置
- 当前环境可用uv.

## 运行要求
- ino / cpp 等文件修改后需要进行编译验证, 确保没有错误.
- 子agent最多同时运行5个, 超过5个则以5个为一批, 逐批运行.

## 设计规范
- 由于设备屏幕尺寸限制, 所有字体大小不能小于1.0.

## 特殊文件

### 临时文件
- 临时脚本与临时生成的文件放到 temp 目录下.

### 数据库文件
- 对数据库的改动需要经过我的同意
- 数据库备份文件存放在 backup 目录下

## 工具
- 编译可使用 scripts/build.ps1 脚本.
- 上传可使用 scripts/upload.ps1 脚本.
